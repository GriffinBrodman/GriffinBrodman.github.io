Lockheed Martin

Bash, C++, Hibernate, Java, JPA, MySQL


"Headquartered in Bethesda, Md., Lockheed Martin is a global security and aerospace company that employs about 112,000 people worldwide and is principally engaged in the research, design, development, manufacture, integration and sustainment of advanced technology systems, products and services" - Lockheed Martin



I spent two summers interning at Lockheed Martin. For my first summer, I worked on Technical Operations. I worked on an Event-Driven System Simulation. I implemented software updates in C++ to increase code maintainability, and reduce redundancy. I wrote Bash scripts to maintain our development environment, and created an automatic documentation generator for our code base. A security clearance was required.


In my second summer, I worked as a full stack developer using Linux server for production and a Windows environment for development. I used Java Persistence API (JPA) with the Hibernate Engine for the data model on top of MySQL, and developed a user interface using Eclipse RAP framework to deliver a browser based solution on a closed network. I served as a Full Lifecycle developer responsible for interfacing with customers and implementing requirements. A security clearance was required.
