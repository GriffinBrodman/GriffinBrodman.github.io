Cloud

Actionscript, Flixel


As a happy go lucky cloud, explore the world while raining on the beings below. Some will be extraordinarily greatful for the rain, while others may not be so cheery. However, be careful not to metaphorically, or physically, drown others in your generosity.

I did this project with my friend Max as part of the 0hr Game Jam. We made this game in the extra hour gained by daylight savings time. Max provided all the artwork as I coded the game. It was super interesting to code under the constraint of having only an hour, it changed a lot of the usual development process. Cloud ended up being an art game about the power of abundance and not having "enough".
